# RapidCR
 RapidCR, a web-based Crisis Response Platform for hospitality environments. It facilitates real-time emergency coordination through secure, role-based interfaces, featuring active incident reporting, live tracking, and data export.
 
PROJECT NAME:- RapidCR[Unified Crisis-Response Platform]


PROBLEM STATEMENT:-Fragmented emergency response: civilians, responders, and coordinators work on disconnected tools, causing delayed dispatch, duplicate data entry, and information leaks across roles.


SOLUTION:-RapidCR is a single web platform that closes the loop between citizens reporting emergencies and the agencies responding to them.RapidCR serves as a **dynamic intelligence layer** that replaces fragmented legacy systems with a unified digital ecosystem. By integrating high-fidelity data—such as automated GPS pings, live video streams, and real-time sensor data—directly into the agency’s dashboard, it eliminates the guesswork inherent in traditional voice dispatch.
This transparency transforms the citizen from a passive bystander into an **active sensor**, providing responders with a tactical advantage before they even arrive. By synchronizing cross-agency workflows and automating status updates back to the reporter, RapidCR ensures accountability, reduces response latency, and fosters a sense of communal safety through reliable, two-way communication.


FEATURES:-Big SOS Button, Quick-Dial Number, Nearest Help Finder, Incident Reporting, Coordinator Dashboard, Responder Hub, Volunteer Portal, Admin Consol, Hard Role Isolation


TECH STACK:-TanStack Router, Framer Motion, Cloudflare Workers, Google Gemini, Claude, Lovable Ai, Git+GitHub, Cloud deploy, PostgerSQL, Lovable Cloud.
